# GitHub Repository Health Analyzer

A web-based tool to analyze the health and maintainability of public GitHub repositories.

## Features

- **Activity Metrics**: Tracks commit recency, frequency, and contributor count
- **Community Health**: Analyzes issue closure rates and response times
- **Project Maturity**: Evaluates repository age, releases, and star count
- **Documentation Score**: Checks for README, license, wiki, and homepage
- **Recommendations**: Provides actionable insights based on analysis
- **Export**: Generate printable health reports

## Usage

1. Open `index.html` in your browser
2. Enter a GitHub repository (e.g., `facebook/react`)
3. Optionally add a GitHub token for higher rate limits
4. Click **Analyze Repository** to see the health report

## Tech Stack

- HTML5 / CSS3 / Vanilla JavaScript
- GitHub REST API
